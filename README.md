# Vanity Address API Service



## Project setup

```bash
pnpm install

pnpm run build

pnpm run start

# or start with pm2
pm2 start pnpm --name "vanity-api-service" -- start
```

## Compile and run the project

```bash
# development
pnpm run start

# watch mode
pnpm run start:dev

# production mode
pnpm run start:prod
```

## Run tests

```bash
# unit tests
pnpm run test

# e2e tests
pnpm run test:e2e

# test coverage
pnpm run test:cov
```
